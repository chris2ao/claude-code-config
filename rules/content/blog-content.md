# Blog Content Rules

## No Private Repository Links

NEVER link to private GitHub repositories in blog posts. Readers cannot access them, so the links are broken for everyone except the repo owner.

**Public repos (safe to link):**
- `chris2ao/cryptoflexllc`
- `chris2ao/claude-code-config`

**All other `chris2ao/*` repos are private.** Do not link to them.

When mentioning a private repo:
- Use the repo name as inline code (e.g., `CJClaude_1`) without a link
- Or describe what it contains without referencing the URL
- Never create clickable links to private repos
